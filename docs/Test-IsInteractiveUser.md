# Test-IsInteractiveUser

## SYNOPSIS
Tests if the current user is interactive.

## SYNTAX

```
Test-IsInteractiveUser [<CommonParameters>]
```

## DESCRIPTION
Tests if the current user is interactive.

## EXAMPLES

### EXAMPLE 1
```
Test-IsInteractiveUser
```

Return $true if the current user is interactive, $false otherwise.
